TASK 1 — FIX BROKEN SCHEMA JSON ON INDUSTRY PAGES
===================================================

Some industry pages have invalid JSON-LD schema causing a parse error:
"Expecting property name enclosed in double quotes" at approximately position 1558.

The cause is a trailing comma inside the BreadcrumbList or Organization object — 
a comma after the last property before the closing brace, like this:
  "url": "https://greyradius.com"
  },    ← trailing comma here
},

Step 1: Find the template file used to generate industry pages. It will contain
a multi-item JSON-LD array with Organization, Service, BreadcrumbList, FAQPage types.

Step 2: Find every instance of:
  },
  },
or
  "url": "https://greyradius.com"
    },
  },

where the inner closing brace has a trailing comma before the outer closing brace.

Step 3: Remove the trailing comma. The correct pattern is:
  "url": "https://greyradius.com"
    }
  },

Step 4: Validate the corrected JSON by running:
  node -e "JSON.parse(require('fs').readFileSync('template-file.html','utf8').match(/<script type=\"application\/ld\+json\">([\s\S]*?)<\/script>/)[1])"

Step 5: Re-generate or patch all existing industry pages that were built from the
broken template. Specifically confirm philippines-market-entry.html and

TASK 3 — WWW TO NON-WWW 301 REDIRECT
===================================================

https://www.greyradius.com/ currently returns HTTP 200 instead of redirecting.
This creates a duplicate content risk.

Add a server-level 301 redirect from all www.greyradius.com requests to greyradius.com.

If the project uses Express:
  app.use((req, res, next) => {
    if (req.headers.host.startsWith('www.')) {
      return res.redirect(301, 'https://greyradius.com' + req.url);
    }
    next();
  });

If using Nginx, add to the www server block:
  return 301 https://greyradius.com$request_uri;

If using a static host configuration file, add the redirect rule there.

Test: curl -s -o /dev/null -w "%{http_code} %{redirect_url}" https://www.greyradius.com/
Must return: 301 https://greyradius.com/
